\# Netflix Content Analyser

A data analysis project experimenting netflix's content library using python, pandas and matplotlib



\## Dataset

8,807 Netflix titles sourced from Kaggle, covering movies and TV shows added up to 2021.



\## Key Findings

\- Netflix's library is 70% Movies and 30% TV Shows

\- The United States produces the most content by far with 3,689 titles

\- Netflix's biggest growth year was 2019 with 1,999 titles added

\- International Movies and Dramas are the most common genres

\- The average Netflix movie runtime is 99.6 minutes



\## Skills Demonstrated

\- Data cleaning and handling missing values with pandas

\- Exploratory data analysis on a real world dataset

\- Data visualisation with matplotlib
